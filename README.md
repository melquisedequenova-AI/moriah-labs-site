# Moriah Labs

> Boutique de tecnologia · Automação, sistemas web, sites e identidade visual.

Site institucional da Moriah Labs AI — construído com Vite + React.

## Stack
- React 18 + Vite
- CSS customizado (moriah.css + moriah-sections.css)
- Deploy via Vercel

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
